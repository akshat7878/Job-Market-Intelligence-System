# Job Market Intelligence System

## Overview
A Data Engineering project that analyzes LinkedIn job market trends using Python, SQL, SQLite, and ETL pipelines.

## Dataset
LinkedIn Job Postings Dataset

## Technologies
- Python
- Pandas
- SQLite
- SQL
- Git
- Docker

## Database Schema
companies
postings
skills
industries
job_skills
job_industries
salaries

## Features
- ETL pipeline
- Data validation
- Skill demand analysis
- Salary analysis
- Industry analysis
- Company hiring analysis
